## Detail Pengerjaan

Ketentuan:
- Buatlah fitur darkmode dan light mode (jika awalnya light mode dapat berubah ke dark mode begitu juga sebaliknya)
- Lakukan manipulasi DOM minimal 2 style selain warna pada darkmode
- Gunakan salah satu fitur pop up box
- Gunakan addEventListener
- Poin plus jika ada tambahan event menggunakan jQuery

Pengerjaan:
- Di navbar, sebelah kanan
- Pada darkmode, card memiliki `border-radius` dan beberapa `section` memiliki perubahan `background`
- `alert` ketika pertama kali masuk
- Pada tombol darkmode
- Coba klik `Upload Kertas-mu`
